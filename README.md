# Game-Main-Menu
I made a Horror futuristic game main menu
This interface is designed for the players who want to play the game
The user should be able to understand that this is the game main menu, and that this is how they get into the game.
I mainly took inspiration from Cyberpunk, and Horror game Main menu's wanting the futuristic style but also a sense of eerieness
Layout: Title at the top with several blocks underneath, showing playercount, ping, level progression (system integrity) and what difficulty theyre playing on
Color: Mainly uses shades of blue, white, and a bit of orange in the background image
Typography: Primary Font: 'Courier New', Fallback Fonts: Courier, monospace, sans-serif
Visual hiearchy: Title, content card, play button, settings, credits, and exit buttons, and then the footer.
Feedback: I have made the text and size of the screen adjustable so that it can show up much easier on smaller devices and screens. 
Responsive design: Everytime a button is hovered over or pressed, the button lights up.
